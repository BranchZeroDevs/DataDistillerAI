# Changelog

All notable changes to DataDistiller AI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-02-12

### Added
- Complete rewrite of README with professional structure
- MIT License
- Contributing guidelines (CONTRIBUTING.md)
- Code of Conduct (CODE_OF_CONDUCT.md)
- Python package configuration (pyproject.toml)
- GitHub Actions CI workflow
- Installation verification script
- Quick start example
- Comprehensive badges and shields
- Professional project structure

### Changed
- Improved README with clearer value proposition
- Better organized documentation
- Enhanced .gitignore for cleaner repository
- Streamlined project focus on V1 (local RAG)

### Removed
- Redundant documentation from archive folder
- app_simple.py (consolidated into main app.py)
- Outdated setup scripts

## [0.9.0] - Previous Version

### Features
- Local RAG system with Ollama integration
- Knowledge Graph visualization (4 modes)
- Multi-LLM support (Ollama, Claude, Gemini)
- Streamlit web interface
- Document processing (PDF, DOCX, TXT, HTML, MD)
- FAISS vector store integration
- spaCy NLP and entity recognition
- NetworkX graph analysis
- V2 production architecture (experimental)
  - FastAPI REST API
  - Kafka message queue
  - PostgreSQL metadata storage
  - MinIO object storage
  - Async worker pools
